Lecture 2: HTML Basics

In this lecture, we will cover the basic structure of an HTML document, including elements such as headings, paragraphs, and links.

HTML Document Structure

An HTML document starts with a doctype declaration followed by the <html> tag. Inside the <html> tag, we have the <head> and <body> sections.

Head Section

The head section contains metadata about the document, such as the title and character set.

Body Section

The body section contains the content that is displayed in the browser.
